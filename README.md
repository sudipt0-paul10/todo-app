# Todo App

This project contains a simple **Todo application** that allows users to manage their tasks efficiently. The application follows a **full-stack architecture** with a React frontend, an Express backend, and MongoDB for database storage.

## Features

-   Create a new todo
    
-   View existing todos
    
-   Mark a todo as completed (update status)
    

## Tech Stack

**Frontend**

-   React
    
-   JavaScript
    
-   HTML / CSS
    

**Backend**

-   Node.js
    
-   Express.js
    

**Database**

-   MongoDB
